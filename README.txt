הוראות להפעלת התוכנה לצורך בדיקת הפרויקט:

•	התקינו את תוכנת .STUDIO 3T 
•	פתחו באמצעות STUDIO 3T את קובץ הגיבוי של ה-DB (באפשרות של BSON archive).
•	כעת פתחו ב-visual studio code את התיקיה  BACKEND, בצעו npm install, רישמו בטרמינל npm start, ולחצו ENTER.
•	כעת פתחו ב-visual studio code את התיקיה FRONTEND, בצעו npm install, רישמו בטרמינל npm start, לחצו ENTER, לאחר מכן בחרו Y.
•	כעת הפרויקט פועל ומורץ בדפדפן ותוכלו להתרשם (התיעודים בגיבוי ה-DB הם מהתאריכים 6-7 ליולי).
